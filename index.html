About PBR Material Reference

This tool provides PBR material reference values for albedo, roughness, metallic, specular and reflectance.

It can be used for Unreal Engine, Unity, Blender, and generic PBR workflows.

Supported materials include concrete, wood, metal, plastic, glass, fabric, brick, asphalt, snow, skin and more.
