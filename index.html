About this tool

This PBR material reference tool provides realistic starting values for physically based rendering workflows.

You can find correct albedo, roughness, metallic, specular, and reflectance ranges for common materials such as concrete, wood, metal, plastic, glass, fabric, snow, skin, and more.

Supports Unreal Engine, Unity, Blender, and generic PBR pipelines.
